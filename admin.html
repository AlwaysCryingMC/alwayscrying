发布 / 编辑文章

用 GitHub Token 在浏览器里直接发文章，写进你的仓库。Token 只存在你这台浏览器里，只发往 GitHub，不经过任何第三方。

怎么生成 Token：GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → 仅选 alwayscrying 仓库 → Permissions 里 Contents 选 Read and write；要管理评论再把 Discussions 也设为 Read and write。详见 README。
